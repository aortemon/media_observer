## Описание

Программа предоставляет возможность мониторинга медиа файлов по заранее заданным расширениям с регулярным обновлением по интервалу. Сведения мониторинга доступны в json формате по адрессу [http://localhos:1234/media_files](http://localhos:1234/media_files)

## Сборка и запуск

Чтобы собрать проект, выполните следующие команды:

```bash
    mkdir build && cd build
    cmake ..
    cmake --build .
```

Эти команды создадут исполняемый файл `main` в директории `build/`.

Запуск программы выполняется командой:

`./main <интервал> <директория>`

`<интервал>` - периодичность обновления списка медиафайлов в секундах
`<директория>` - путь к директории для мониторинга
